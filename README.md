# Steam Friends Integration for Home Assistant

A Home Assistant integration that shows your Steam friends list using a **single API key**.

## Features
- Single API key for all friends
- Complete friends list with current status
- Auto-refresh every 5 minutes (configurable)

## Installation
Add to HACS as custom repository:
https://github.com/mmynonetheless/ha-steam-friends

text

## Configuration
1. Get Steam API key from https://steamcommunity.com/dev/apikey
2. Find your Steam ID (64-bit number)
3. Add integration via Settings → Devices & Services
